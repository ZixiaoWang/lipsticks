# 口红色号找寻器💄
## 点这 [首页](https://zixiaowang.github.io/lipsticks/) 开启口红色号找寻之旅。
## 👉 这是啥？
钢铁直男必备，可以通过口红品牌直达色号，或者通过颜色直达品牌。
